Outfit Scoring System

Multi-factor scoring (max 105 pts): Color Harmony (30), Skin Flattery (20), Body Shape (20), Weather (20), Occasion (10), + Trending Bonus (5).

Color Theory Engine

Implements color wheel relationships: complementary (180°), analogous (±30°), monochromatic (same hue). Calculates harmony scores for outfit scoring.

Outfit Constraints

Defines outfit structure rules: requires top+bottom OR dress (never both), adds outerwear when cold/raining, ensures formal occasions have polish.
